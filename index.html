View Details
Introduction

This project focuses on the development of a MATLAB-based system designed to accurately classify human emotions like happiness and anger from audio samples. It leverages digital signal processing to analyze vocal characteristics, providing an automated solution for emotion detection in smart devices.

The system processes raw audio by analyzing frequency parameters like pitch and energy levels. It employs Mel-frequency cepstral coefficients (MFCC) to capture spectral features of the voice. These features are then compared against pre-defined statistical profiles to determine the likely emotional state.

I2C and SPI Communication

Implementation of multi-node Master/Slave protocols.

View Details
Introduction

Designed a robust communication bridge between multiple microcontrollers using standard industry protocols. The project explores the speed, distance, and reliability trade-offs between I2C and SPI, allowing for a multi-node sensor network to report back to a single master controller.

Working

The master device initiates clock pulses and data addresses via the I2C bus to find slaves. Slave devices respond with sensor data packets using the high-speed synchronous SPI interface. Logic level converters ensure voltage compatibility across different hardware while preventing data signal collisions on the bus lines.

My Role

I designed the wiring architecture to ensure perfect protocol synchronization across nodes. I wrote the Master/Slave firmware in Embedded C to manage the data buffers effectively. I verified data integrity using an oscilloscope during signal transmission and optimized the baud rate for stable transfers.

Smart Plant Watering

Automatic irrigation system using soil moisture detection.

View Details
Introduction

Created an automated irrigation system designed to optimize water usage in domestic gardening and agriculture. It monitors soil conditions in real-time to prevent plant dehydration, reducing manual labor while significantly improving overall plant health and growth efficiency.

Working

Soil moisture sensors measure the dielectric constant of the earth to determine current wetness. When moisture falls below a pre-set threshold, a relay is triggered to activate a submersible water pump. Once the sensor detects saturation, the system automatically cuts power to stop the water flow.

My Role

I integrated the sensor array and built the relay-based pump control circuit from scratch. I calibrated the moisture threshold levels for different plant types to ensure optimal watering. I also designed the power supply circuit to safely handle both 5V logic and 12V pump components.
